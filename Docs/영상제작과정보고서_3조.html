1. 프로젝트 개요

중국 동북 지역 어학연수 경험을 바탕으로 중국 학교 문화의 특징을 한국과 비교하는 90초 분량의 영상을 제작했다. 주요 비교 항목은 등교 시간, 국기게양식(升旗仪式), 교복 + 빨간 스카프(红领巾), 눈 보호 체조·라디오 체조, 대학 신입생 군사훈련(军训)이다.

핵심 메시지는 “국가 주도 집단 교육의 차이”이며, 픽사 스타일 3D 애니메이션으로 시각화했다.

2. 사용 툴 및 제작 파이프라인

  • 이미지 생성: Gemini, ChatGPT
  • 영상화: Kling AI (Image-to-Video)
  • 보조 툴: Google Flow
  • 음성·자막: Clova Note / TTS + Vrew
  • 편집: Vrew

제작 흐름: 캐릭터 레퍼런스 시트 제작 → 앵커 프롬프트 확정 → 씬별 이미지 생성 → Kling AI 클립 변환 → TTS·자막 작업 → Vrew 합본

3. 캐릭터 일관성 확보 전략

AI 이미지 생성의 최대 약점인 캐릭터 불일치를 해결하기 위해 다음 방법을 적용했다.

① 캐릭터 레퍼런스 시트 사전 제작
한국 학생과 중국 학생 각각 정면·측면·후면 이미지를 먼저 생성했다. 두 캐릭터 모두 짧은 검정 단발로 통일했다.

② 앵커 프롬프트 고정 적용
모든 이미지 생성 프롬프트 앞부분에 아래 문장을 고정으로 사용했다:

Pixar-style 3D animated characters, two students: Korean student — navy blue school uniform jacket + white shirt, short black bob hair… Chinese student — red tracksuit uniform + red neckerchief (红领巾), short black bob hair… consistent character design, same faces throughout all scenes, expressive eyes, soft cel-shaded 3D render, no text, no UI elements.

4. 사실 검증 과정

영상 내용은 다음과 같이 교차 검증했다:

  • 팀원 중 1명의 중국 동북 지역 직접 체류 경험 (1차 자료)
  • Claude를 통한 내용 정리 및 보완
  • 중국 교육 관련 자료 교차 확인

5. 발생한 문제와 해결 과정

문제 원인 시도한 해결 결과
캐릭터 얼굴·의상 불일치 AI 생성 시마다 변형 레퍼런스 시트 + 앵커 프롬프트 고정 적용 대부분의 씬에서 일관성 확보
[Kling AI 실제 실패 사례를 여기에 작성하세요]

예시:
• 문제: 씬 05 군사훈련에서 행진 모션이 부자연스럽고 다리가 겹쳐 보임
• 발생 씬: 씬 05 (중국 대학 군사훈련)
• 원인 추정: Kling AI가 복잡한 집단 동작을 제대로 처리하지 못함
• 시도한 해결: 모션 프롬프트 수정 + 3회 재생성
• 결과: 여전히 어색하여 Google Flow로 대체 생성
Kling AI 크레딧 부족 무료 플랜 제한 팀원 5명 계정 분산 사용 + Google Flow 병행 클립 생성 지연 최소화
나레이션 타이밍 불일치 TTS 기본 속도 문제 Vrew에서 수동 속도 조절 및 자막 싱크 청취 편의성 확보

6. 팀원 역할 분담

  • 김다현 (팀장): 기획 총괄, 나레이션 스크립트 확정, TTS 생성, 자막 제작, 발표 준비
  • 조은선: Kling AI 영상화 총괄 및 Vrew 최종 편집·합본
  • 조용민: 씬 01·02 이미지 생성 + 캐릭터 레퍼런스 시트 제작
  • 정성훈: 씬 03·04 이미지 생성
  • 최성빈: 씬 05 + 오프닝·클로징 이미지 생성

7. 결론

이번 프로젝트는 AI 도구의 한계를 보완하기 위한 프롬프트 전략과 팀 협업이 핵심이었다. 캐릭터 레퍼런스 시트와 앵커 프롬프트를 통해 일관성을 확보하려 노력했으며, 실제 Kling AI 사용 과정에서 발생한 기술적 한계는 팀원 간 신속한 대체 툴 활용으로 대응했다.
