Step 3 — 전체 제작 워크플로우

1
캐릭터 생성
Midjourney 또는 Krea → 레퍼런스 시트 (정면/전신/배경없음) 출력
2
씬별 이미지 생성
캐릭터 이미지 + 씬 프롬프트 → Krea / Midjourney / Ideogram으로 각 씬 이미지 제작
3
Kling AI 영상화
씬별 이미지 업로드 + 영상 프롬프트 입력 → 각 5초 클립 생성 (Image-to-Video 모드)
4
TTS 음성 제작
CLOVA Dubbing / ElevenLabs → 대사 2줄 녹음 후 씬에 맞게 싱크 조절
5
클립 합치기
CapCut / DaVinci Resolve → 씬1(5초) + 씬2(5초) 연결, 음악+TTS 합성, 컬러 그레이딩
